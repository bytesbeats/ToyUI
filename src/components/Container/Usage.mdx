# Getting Started

The `Container` component is a simple wrapper component that can be used to wrap other components. It is useful for centering content on the page.

```jsx
import { Container } from '@cuteToy/blocks';

const App = () => (
  <Container center>
    <h1>Hello, world!</h1>
  </Container>
);
```
