## Getting Started

The `Icon` component is a simple wrapper component that can be used to wrap other components. It is useful for centering content on the page.

## Usage

```jsx
import { Icon } from '@cuteToy/blocks';

const App = () => (
  <Icon name="heart" />
);
```
